# Personal Portfolio

A minimal, modern portfolio website built with Next.js 14, React, TypeScript, and Tailwind CSS.

## Features

- Modern tech stack (Next.js, React, TypeScript)
- Responsive design
- Dark/Light mode with system preference support
- Clean and minimal UI

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Tech Stack

- Next.js 14
- React
- TypeScript
- Tailwind CSS
- next-themes
